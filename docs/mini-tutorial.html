ZHUANG-AI mini 手机端使用教程

ZHUANG-AI mini 是手机端 Web / PWA 轻量应用，可在手机浏览器中打开，也可以添加到主屏幕使用。它支持主图上传、选区框选、参考图整理、提示词预设、GRS API 生成、对话分析、历史记录和本地设置保存。

适用：中国常见手机浏览器 · Web / PWA

建议优先使用手机系统自带浏览器、Chrome、Edge、QQ 浏览器、夸克浏览器或 UC 浏览器。不同浏览器菜单名称会略有差异，但核心步骤都是：打开 mini/ 页面 → 允许相册权限 → 添加到桌面或主屏幕。

一、快速开始

  1. 在手机浏览器里打开官网里的 ZHUANG-AI mini 入口，或打开本教程同级目录下的 mini/。
  2. 第一次使用先进入底部 设置，填写 GRS API Key 和 GRS API 地址。
  3. 回到 创作 页面，点击 选择图片 添加主图。
  4. 在主图上拖拽框选要处理的区域，点击 读取选区。
  5. 按需要添加最多 4 张参考图，选择生图模型、图片大小和生成数量。
  6. 选择提示词预设或自己输入提示词，点击 开始生成。
  7. 生成结果会显示在页面下方，同时保存到本机浏览器的 历史 里。

二、核心功能怎么用

主图与选区

点击 选择图片 从相册选择主图。图片出现后，直接在图片上按住拖拽框选区域；需要精确比例时，可以开启 锁定比例 并选择比例。

参考图

参考图最多 4 张，适合作为脸部、服装、发型、材质、构图或风格参考。添加后可预览，也可以删除后重新选择。

模型与输出

生图模型包含 Nano Banana 系列和 gpt-image-2 / claude-imagine-2。图片大小可选自适应、1K、2K、4K，生成数量可选 1 到 4 张。

提示词预设

在提示词区域可以搜索预设、按分类筛选，也可以手动输入自己的修图目标。常用提示词可保存为自己的预设。

对话分析

进入底部 对话，选择 GRS 对话模型后输入问题。它可以用来分析修图方向、优化提示词、整理后期步骤。

历史与设置

历史 和 设置 都保存在当前浏览器本机数据里。换浏览器、清理缓存或使用无痕模式，可能会导致记录和 API Key 消失。

三、添加到桌面 / 主屏幕通用说明

  1. 先用手机浏览器打开 ZHUANG-AI mini 页面。
  2. 点击浏览器底部或右上角的 菜单、更多、分享 或 三点 按钮。
  3. 找到 添加到桌面、添加到主屏幕、安装应用、创建快捷方式 或类似选项。
  4. 确认名称为 ZHUANG-AI mini，点击添加。
  5. 回到手机桌面，点击图标即可像 App 一样打开。

如果看不到添加入口，先确认当前不是无痕模式，并尝试刷新页面或换用系统浏览器、Chrome、Edge、QQ 浏览器、夸克浏览器。

四、中国常见浏览器教程

下面覆盖国内用户常见的手机浏览器。不同版本界面可能会变化，如果菜单名称不同，优先寻找含有“桌面、主屏幕、安装、快捷方式、收藏”的入口。

1. 微信内置浏览器

  1. 在微信里打开 mini 页面。
  2. 点击右上角 ···。
  3. 如需稳定使用，选择 在浏览器打开，再用系统浏览器或 QQ 浏览器继续。
  4. 微信内置浏览器通常不适合作为 PWA 安装入口，建议只用于临时访问。

2. QQ 浏览器

  1. 打开 mini 页面。
  2. 点击底部或右上角 菜单。
  3. 选择 工具箱 或 添加到。
  4. 点击 添加到桌面，确认添加。

3. 夸克浏览器

  1. 打开 mini 页面。
  2. 点击底部 菜单。
  3. 找到 添加到桌面 或 工具箱 里的桌面快捷方式。
  4. 确认后从手机桌面打开。

4. UC 浏览器

  1. 打开 mini 页面。
  2. 点击底部 菜单。
  3. 进入 收藏/历史、工具箱 或 添加 相关入口。
  4. 选择 添加到桌面 或 创建快捷方式。

5. 百度浏览器 / 百度 App

  1. 打开 mini 页面。
  2. 点击底部或右上角的 菜单。
  3. 如果有 添加到桌面，直接添加。
  4. 如果只有收藏入口，可先收藏；需要 PWA 桌面图标时建议换系统浏览器或 Chrome。

6. 360 浏览器

  1. 打开 mini 页面。
  2. 点击 菜单 或 三横线。
  3. 找到 添加到桌面、保存到桌面 或 网页快捷方式。
  4. 确认添加后，从桌面图标进入。

7. 搜狗浏览器

  1. 打开 mini 页面。
  2. 点击浏览器 菜单。
  3. 选择 收藏 或 添加到桌面。
  4. 如果没有桌面入口，可使用收藏访问，或换支持 PWA 更完整的浏览器。

8. 小米浏览器

  1. 打开 mini 页面。
  2. 点击底部 菜单。
  3. 选择 添加到桌面 或 桌面快捷方式。
  4. 如系统提示权限，允许浏览器创建桌面快捷方式。

9. 华为浏览器

  1. 打开 mini 页面。
  2. 点击底部或右上角 菜单。
  3. 选择 添加至桌面、添加到主屏幕 或 收藏。
  4. 如果系统拦截，进入手机设置允许浏览器创建桌面快捷方式。

10. OPPO 浏览器

  1. 打开 mini 页面。
  2. 点击 菜单。
  3. 选择 添加到桌面 或 桌面快捷方式。
  4. 按系统提示开启快捷方式权限。

11. vivo 浏览器

  1. 打开 mini 页面。
  2. 点击 菜单。

  4. 添加后回到桌面检查图标。

12. 荣耀浏览器

  1. 打开 mini 页面。

13. 三星浏览器

  4. 选择 主屏幕，确认添加。

  1. 用 Safari 打开 mini 页面。
  2. 点击底部 分享 按钮。
  3. 向下滑动，选择 添加到主屏幕。

15. Chrome

  2. 点击右上角 三点。
  3. 选择 安装应用、添加到主屏幕 或 添加到桌面。

16. Edge

17. Firefox

18. 2345 浏览器

五、权限说明

相册 / 文件权限

选择主图和参考图时，浏览器会请求访问相册或文件。请选择允许，否则无法上传图片。

桌面快捷方式权限

部分安卓系统会禁止浏览器创建桌面图标。进入系统设置，找到对应浏览器，允许 创建桌面快捷方式 或 桌面权限。

本地存储

API Key、历史记录、预设和设置保存在当前浏览器本地。不要在别人的手机上保存自己的 API Key。

网络权限

生图和对话需要访问你填写的 GRS API 地址。网络不稳定、地址错误或 Key 错误都会导致请求失败。

六、更新与缓存

  1. 如果官网更新了 mini，但你看到的还是旧页面，先下拉刷新或点击浏览器刷新。
  2. 如果仍然没更新，进入浏览器设置，清理当前站点缓存后重新打开。
  3. 已经添加到桌面的 PWA 图标也可能缓存旧版本，可以删除桌面图标后重新添加。
  4. 清理缓存前请确认自己还记得 GRS API Key，因为清理后本地设置可能会丢失。

七、常见问题

1. 点“选择图片”没有反应

检查浏览器是否有相册或文件权限；如果在微信内置浏览器里失败，选择“在浏览器打开”后再试。

2. 无法添加到桌面

先检查浏览器是否支持桌面快捷方式，再检查手机系统是否禁止该浏览器创建桌面图标。部分浏览器只能收藏，不能完整安装 PWA。

3. 生图失败

进入 设置 检查 GRS API Key、GRS API 地址和网络连接。API Key 错误、余额不足、接口地址填错都会失败。

4. 历史记录不见了

历史记录保存在本机浏览器。换浏览器、无痕模式、清理缓存、卸载浏览器或删除站点数据都会让历史记录消失。

5. 页面显示错乱

刷新页面，或清理当前站点缓存后重开。如果手机系统浏览器版本太旧，建议换 Chrome、Edge、QQ 浏览器或夸克浏览器。
