ZHUANG-AI Core

已完成版本的功能与优点

当前可下载的 ZHUANG-AI 主推模型接入和 Photoshop 工作流融合：图像生成走 gptimagine-2 与 Nano Banana 全系列，文字对话走 Imagine-2 Imagine-2 与 Gemini 系列，并通过选区、参考图、预设和批处理把它们落到真实修图场景。

01

选区图生图

直接读取 Photoshop 当前框选区域作为主图，生成结果可回写到当前文档，普通对齐不会改变画布大小。

02

多参考图

主图与参考图分离，最多 4 张参考图，可作为风格、脸部、材质和细节参考，支持预览、删除和重新抓取。

03

GRS 图像接口

支持 Nano Banana-pro、Nano Banana-fast、Nano Banana 2、gptimagine-2 等图生图模型入口。

04

GRS 对话分析

统一使用 GRS 对话模型，支持文字问答，也可以读取 Photoshop 选区进行图文分析和修图建议。

05

预设与批处理

提示词预设可保存参考图；批处理任务会记住添加时的选区和参考图上下文，适合连续生成。

06

SD WebUI 工作流

连接本地 WebUI，读取模型和 Lora，集中管理正向、反向提示词、采样方法、步数和重绘幅度。

Workflow

从 Photoshop 选区到可用结果

ZHUANG-AI 系列的核心逻辑是让 AI 生成尽量贴近真实修图动作，而不是脱离原生 Photoshop 工作流单独出图。

读取选区

在 Photoshop 里框选要修的区域，插件读取主图和选区位置。

添加参考图

继续框选脸部、材质、风格或细节区域，作为额外参考图。

选择模型与预设

选择 GRS 图生图模型，套用提示词预设或输入自定义修图目标。

生成并回写

生成完成后自动放回 Photoshop 当前文档，继续人工精修。

Changelog

版本历史

1.3.8 正式版（2026-05-16）
  • 修复 GRS 图生图轮询成功但无图时被误判为完成的问题，避免继续报“无法从响应中提取图像”。
  • 补强任务 ID 与结果提取兼容范围，适配更多返回结构并改进失败原因显示。
  • 修复预设在 UXP 文件系统尚未就绪时被提前缓存为空的问题，首次启动即可正常读取 yushe.json。
  • 保留批处理任务的参考图上下文，并同步教程、发布包与官网版本号到 1.3.8。
1.3.4 公测版
  • 接入 GRS /v1/chat/completions 文字生成接口，对话功能统一使用 GRS API 密钥。
  • 清理 OpenAI、NewAPI、Google 等非 GRS 对话模型入口。
  • 支持对话读取 Photoshop 选区后进行图文问答。
  • 重构生图、设置、日志等界面结构，提升窄面板可用性。
