1.4.1 重点升级了小应用工作区：VFX 特效、合成辅助器、特效迁移、AI超清、SD WebUI 和 RunningHub 应用集中在“应用/小应用”界面里使用，并统一改为伪下拉框、普通图层混合模式、自然比例生成方案。

一、安装

直接安装

  1. 退出 Photoshop。
  2. 解压 ZHUANG-AI 1.4.1.zip。
  3. 把整个 ZHUANG-AI 1.4.1 文件夹复制到 Photoshop 的 Plug-ins 目录。
  4. 重新打开 Photoshop，在顶部菜单 插件 / Plugins 中打开 ZHUANG-AI 修图插件。

UXP Developer Tool

  1. 打开 UXP Developer Tool。
  2. 点击 Add Plugin。
  3. 选择 ZHUANG-AI 1.4.1/manifest.json。
  4. 点击 Load 或 Reload。

不要只复制 index.html，必须保留 manifest.json、index.html、index.js、yushe.json 和 icons 文件夹。

二、第一次使用前设置 API

  1. 进入插件顶部 设置。
  2. 填写 GRS 生图 API 密钥和 GRS 对话 API 密钥。
  3. 如果使用 NewAPI，填写 NewAPI 地址与密钥。
  4. 如果使用 SD WebUI，填写本地地址，常见为 http://localhost:7860。
  5. RunningHub 应用需要填写 RunningHub API Key。
  6. 点击 保存全部设置。

1.4.1 的公告与更新检查默认读取服务器 https://www.syyyy.online。

三、新版应用/小应用界面

点击顶部 应用 进入小应用工作区。这里集中显示 VFX 特效、SD WebUI、合成辅助器、特效迁移、AI超清和导入的 RunningHub 应用。

  • 所有主要下拉框已改为伪下拉框，避免 Photoshop UXP 原生 select 显示异常。
  • 小应用生成结果默认使用正常混合模式，不再使用滤色。
  • 所有图片生成默认不强制 1:1 或固定区域大小，按当前选区/源图自然比例生成。
  • WebUI 页面提供 返回小应用 按钮，可回到应用首页。

四、VFX 特效使用方案

  1. 在 Photoshop 中框选要作为主图的区域，进入插件读取选区。
  2. 进入 应用 → VFX 特效。
  3. 选择特效预设、材质预设、颜色、粒子、烟雾和运动轨迹描述。
  4. 可点击 添加当前选区为参考图，最多添加 4 张参考图。
  5. 如果需要方向/动势，读取一张轨迹图作为 VFX 轨迹参考。
  6. 点击生成按钮，插件会把 VFX 提示词写入生图区并调用当前生图模型。

VFX 不再限制正方形比例，也不强制缩小到局部小区域；提示词会要求效果按画面需要自然扩展，同时保留主体和构图。

五、合成辅助器使用方案

合成辅助器用于从原图直接生成后期合成常用的辅助图，例如深度图、法线图、分割图、雾效遮罩、高光反射图、边缘线稿和亮度图。

  1. 进入 应用 → 合成辅助器。
  2. 点击 读取当前选区，把当前 Photoshop 选区作为原图。
  3. 勾选需要输出的辅助类型，可多选。
  4. 选择文字/分析模型和生图模型；默认可不启用文字分析，直接用专用提示词 + 原图生成。
  5. 选择生图清晰度和生成参数。
  6. 点击 生成辅助图，每个 pass 会生成一张图并回写 Photoshop。

合成辅助器的专用提示词会要求不改变构图、不改变主体结构、不添加文字或水印。

六、特效迁移使用方案

方案一：先分析再迁移

  1. 读取原图/当前主体。
  2. 读取特效源图/参考图。
  3. 选择 先分析特效图生成提示词，再迁移。
  4. 文字模型只分析特效源图的颜色、材质、边缘、光影、粒子和镜头语言。
  5. 插件把分析提示词交给生图模型，在原图上迁移特效。

方案二：双图直接参考

  1. 读取原图和特效源图。
  2. 选择 原图 + 特效图直接作为参考图迁移。
  3. 生图模型会把原图作为主体/构图/身份来源，把特效图只作为效果、材质、光影风格参考。
  4. 点击 开始特效迁移。

迁移强度可选轻微、平衡、强烈；结果保持正常图层混合模式。

七、AI超清使用方案

  1. 进入 应用 → AI超清。
  2. 读取要超清的 Photoshop 选区。
  3. 选择 2x、3x 或 4x 放大倍数。
  4. 点击 计算切片，插件会按 Nano Banana 2 单张 4K 的能力自动拆分 tile。
  5. 点击 开始 AI超清，每个 tile 用专用超分提示词生成，再按目标位置回贴。

AI超清提示词强调高清化、边缘补细节、tile 边缘连续，禁止换脸、改构图、增删物体。

八、SD WebUI 使用方案

  1. 先启动本地 SD WebUI，并确认开启 API。
  2. 在设置里填写 WebUI 地址。
  3. 进入 应用 → SD WebUI。
  4. 点击刷新读取模型、采样器和 LoRA。
  5. 填写正向提示词、反向提示词、步数、CFG、图片数量。
  6. 重绘幅度 现在是文字输入框，可输入 0 到 1 的小数，例如 0.35。
  7. 点击生成，结果回写 Photoshop。

WebUI 顶部有 返回小应用，可回到应用首页。

九、RunningHub 应用使用方案

  1. 在设置中填写 RunningHub API Key。
  2. 进入 应用 页面，打开 RunningHub 导入/管理区域。
  3. 粘贴 RunningHub 应用链接或 App ID，点击解析/导入。
  4. 导入后会在应用首页出现应用卡片。
  5. 打开应用卡片，图片字段会显示为读取 Photoshop 选区的控件，文本/数值字段按 RunningHub 参数填写。
  6. 点击 运行应用。插件会先上传图片输入，再提交任务并轮询结果。
  7. 返回图像会自动放回 Photoshop。

RunningHub 流程参考小T（PixelRunner / 小T修图助手）的公开思路，项目内 NOTICE 文件保留署名。

十、常见问题

生成失败

先检查 API Key、余额、模型是否可用。RunningHub 的余额不足、队列已满属于服务端状态，需要到对应平台处理。

参考图不显示

重新进入应用页或重新添加当前选区为参考图。1.4.1 已修复参考图数据格式兼容与 VFX 预览刷新。

下拉框遮挡输入框

1.4.1 已将应用里的下拉控件改为内联伪下拉框，展开时会占用布局空间，不再盖住下面的提示词输入框。

比例或区域不对

1.4.1 的小应用生成请求默认使用自然比例，不再强制 1:1。回贴位置仍按源选区或应用输入区域放回 Photoshop。
