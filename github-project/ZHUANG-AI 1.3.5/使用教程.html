ZHUANG-AI 修图插件 1.3.5 使用教程

这是一份尽量简单的教程。照着做，就能把插件装进 Photoshop，并完成读取选区、添加参考图、生成图片、对话和 WebUI 使用。

版本：1.3.5 正式版

一、这个文件夹里哪些东西不能删？

安装插件时，整个 ZHUANG-AI 1.3.5 文件夹都要保留。里面至少要有这些文件：

  • manifest.json：插件说明文件，Photoshop 靠它识别插件。
  • index.html：插件界面。
  • index.js：插件功能代码。
  • yushe.json：提示词预设。
  • icons 文件夹：插件图标。

不要只复制一个 index.html，也不要改文件名。

二、安装到 Photoshop

方法 1：放到 Photoshop 插件目录

  1. 退出 Photoshop。
  2. 把整个 ZHUANG-AI 1.3.5 文件夹复制到 Photoshop 的 Plug-ins 插件目录。
  3. 重新打开 Photoshop。
  4. 在顶部菜单里找 插件 / Plugins，打开 ZHUANG-AI 修图插件。

方法 2：如果你使用 UXP Developer Tool

  1. 打开 UXP Developer Tool。
  2. 点击 Add Plugin。
  3. 选择本文件夹里的 manifest.json。
  4. 点击 Load 或 Reload。
  5. 回到 Photoshop 打开插件面板。

三、第一次使用前先设置 API

  1. 打开插件后，点击顶部的 设置。
  2. 找到 GRS API 密钥，填入你的 GRS key。
  3. 如果你用 SD WebUI，也填好 SD WebUI API 地址，一般是 http://localhost:7860。
  4. 点击 保存全部设置。

没有 API 密钥时，生图和对话功能会失败。

四、最常用：框选区域后生成图像

  1. 在 Photoshop 打开一张图片。
  2. 用矩形选框工具框住你想修的区域。
  3. 回到插件，进入 生图 页面。
  4. 点击 读取选区，看到预览图就说明读取成功。
  5. 选择生图模型、图片大小、生成数量。
  6. 输入提示词，或者选择一个预设。
  7. 点击 生成图像。
  8. 生成成功后，图片会自动放回 Photoshop 当前文档里。

普通对齐不会改变你的画布大小。

五、1.3.5 新功能：添加参考图

参考图的意思是：除了主选区，还可以给 AI 几张额外图片作为风格、脸部、材质或细节参考。

  1. 先框选主图区域，点击 读取选区。
  2. 再在 Photoshop 里框选另一个想作为参考的区域。
  3. 点击 添加当前选区为参考图。
  4. 参考图会显示在 参考图 区域，最多 4 张。
  5. 想删除某张参考图，点击缩略图右上角的删除按钮。
  6. 想重新抓取某张参考图，点击那张参考图缩略图，再用当前 Photoshop 选区替换。

生成时会自动把主图 + 参考图一起发送给 GRS 图生图接口。

六、预设怎么用？

使用预设

  1. 选择 预设分组。
  2. 选择 预设选择。
  3. 提示词会自动填入。

保存预设

  1. 写好提示词。
  2. 输入 预设名称。
  3. 如果已经添加参考图，也会一起保存。
  4. 点击 保存预设。

七、批处理怎么用？

  1. 设置好提示词、模型、选区和参考图。
  2. 点击 添加到批处理。
  3. 可以换选区、换提示词，再继续添加任务。
  4. 进入 批处理 页面。
  5. 点击 开始批处理。

1.3.5 之后，批处理任务会记住添加任务时的参考图，不会因为你后来换了参考图而丢失。

八、对话功能怎么用？

  1. 进入 对话 页面。
  2. 选择 GRS 对话模型。
  3. 输入问题，点击 发送消息。
  4. 如果想让 AI 看图回答，先在 Photoshop 框选区域，再点击 读取选区。
  5. 读取成功后再发送问题，AI 会结合图片回答。

九、WebUI 功能怎么用？

  1. 先启动你的本地 SD WebUI。
  2. 进入插件 设置，确认 WebUI 地址正确。
  3. 进入 WebUI 页面。
  4. 点击 刷新，读取模型和 Lora。
  5. 填写正向、反向提示词和参数。
  6. 点击 一键生成。

十、常见问题

1. 插件打不开

检查文件夹里有没有 manifest.json、index.html、index.js，不要把文件从文件夹里拆出来。

2. 生图失败

先去设置检查 GRS API 密钥，再确认网络可用。

3. 读取选区失败

确认 Photoshop 当前有打开图片，并且真的用选框工具框出了区域。

4. 回图大小不对

默认用普通对齐。如果你想让回图严格贴合选区，去设置里把图片对齐算法改成 缩放图层到选区。
